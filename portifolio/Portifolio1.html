    • Portifólio

    Html
    - HTML é uma linguagem que pode ser utilizada na construção de sites, foi criada em 1993, por Berners-Lee e Dan Connoly, tendo varias versões com o tempo, sendo interpretada por navegadores, ensinada desde o começo do curso.
    css
    CSS ou Cascading Style Sheet é utilizadodo para estilizar escritos em uma linguagem de marcação, por exemplo o HTML é uma linguagem de marcação que pode ser estilizado, o CSS funciona para decorar seu pagina e colocae os elementos no lugar que o criador quer posicionar.
    Bootstrap
    o Bootstrap é um framework front-end que foi desenvolvido por Jacob Thorton e Mark Otto, engenheiros do twitter, o Bootstrap ajuda bastante na responsividade do mobile, ficando mais facil que fazer varias linhas de CSS, e facilita a incrementar carrosel e outras customizações.
    Javascript
    O JavaScript é uma linguagem de programação muito utilizada em paginas na web, e pode interagir com o usuario, como aparecer algo na tela quando precionado algum botão, mensagem alertando um erro de informação, assim mudando a apresentação visual da pagina.
    JQuery
    JQuery é uma biblioteca do JavaScript, no caso a biblioteca mais famosa do JavaScript, interagindo com o html, simplificando os scripts interpretados no navegador do cliente, e atualmente é muito utilizada.
